## Requirements

Before installing, ensure your environment meets these requirements:

- **WordPress** 6.5 or later
- **PHP** 8.0 or later
- Block editor environment (Gutenberg in WordPress core)
- An AI provider:
  - Native WordPress AI credentials (preferred, available on WordPress 7.0+), or
  - OpenAI API key for the fallback path

## Install from GitHub

1. Download the latest release from [GitHub](https://github.com/whole-code/wp-intelligence/releases)
2. Upload the plugin folder to `wp-content/plugins/wp-intelligence`
3. Activate **WP Intelligence** in WP Admin → Plugins

## Post-Activation

After activation, you'll be redirected to the **Welcome** tab with a setup checklist:

1. **Choose your modules** — Enable only the features your site needs
2. **Configure AI** — Set your provider credentials and model
3. **Set up content workflows** — Optionally configure Content Intelligence

## Verify Installation

Navigate to **Settings → WP Intelligence** and confirm:
- The settings page loads without errors
- The version number displays correctly
- All expected module tabs are visible
