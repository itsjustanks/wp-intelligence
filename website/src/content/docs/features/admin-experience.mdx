The Admin Experience module provides a collection of WordPress admin customizations.

## Features

- **Remove WP Logo** — hides the WordPress logo from the admin bar
- **Custom Login Logo** — replace the WordPress logo on the login page
- **Hide Update Notices** — suppress update nag notices for non-admin users
- **Custom Footer** — replace the default admin footer text
- **Menu Reorganization** — clean up and reorder the admin menu
- **Editor Enhancements** — improvements to the block editor experience

## Editor Enhancements

- **Class Autocomplete** — suggests CSS class names in the block Advanced panel
- **Force Fullscreen** — automatically open the editor in fullscreen mode
- **Auto-Open List View** — open the List View panel by default when entering the editor

## Configuration

All options are toggle-based and configured in the Admin Experience section of WP Intelligence settings. Themes can set defaults via the `wpi_admin_experience_defaults` filter.
