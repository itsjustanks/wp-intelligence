## 0.2.0

- Added React-based settings page using `@wordpress/components`
- Added REST API for settings management (`wp-intelligence/v1/settings`)
- Added documentation website (Astro Starlight)
- Added Featured Image AI module with DALL-E generation and title overlay
- Added Content Intelligence content styles (built-in + custom)
- Added AI Chat module for admin-wide conversations
- Added Query Loop Load More module
- Improved module management with category grouping
- Improved provider fallback with environment variable support

## 0.1.0

- Initial release
- AI Composer with prompt-to-page composition
- Block Visibility with conditional display rules
- Content Intelligence (Syndication) with URL/video rewriting
- Admin Experience white-labeling
- Security, Performance, and Resource Hints modules
- WooCommerce conditional asset loading
- WordPress 7.0+ native AI client support with OpenAI fallback
